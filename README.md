# devops-shell-toolkit

A collection of useful Bash scripts for system monitoring, automation, and daily DevOps tasks.

Prerequisites

- Linux / macOS
- Bash shell
- Basic terminal knowledge

Clone the Repository

git clone https://github.com/punithxpm/devops-shell-toolkit.git

cd devops-shell-toolkit/scripts

How to Run Scripts
Step 1: Give execute permission
chmod +x *.sh
Step 2: Run any script
./system_status.sh
